# Gobernanza de datos sensibles — S.N.E.-E.C.O.

Este documento define qué tipos de datos pueden usarse en E.C.O. de forma educativa, experimental y responsable.

## Principio central

E.C.O. puede estudiar datos sensibles solo como material sintético, público, anonimizado, licenciado o curado para investigación. No debe usarse para diagnosticar personas, inferir conciencia humana real, determinar responsabilidad forense ni modificar umbrales estables sin auditoría.

## Clasificación de uso

| Dominio | Permitido | Condicional | Bloqueado |
|---|---|---|---|
| Diagnóstico | ejemplos sintéticos, taxonomías públicas, literatura | datasets anonimizados/licenciados con trazabilidad | diagnosticar personas reales |
| Clínico | conceptos generales y educativos | datos públicos agregados sin identidad | historial médico personal identificable |
| Conciencia humana | modelos teóricos, literatura, lenguaje simbólico | datasets cognitivos no personales | afirmar detección de conciencia real |
| Forense | casos públicos, textos legales abiertos, ejemplos sintéticos | análisis documental sin inferir culpabilidad | atribuir responsabilidad legal a una persona |
| Umbrales | simulación experimental en rama separada | recalibración con reporte, delta y validación | cambiar baseline estable sin auditoría |

## Regla operativa

Un dato sensible solo puede ingresar al pipeline si cumple al menos una condición:

- es sintético;
- es público y legalmente reutilizable;
- está anonimizado;
- tiene licencia clara;
- no identifica personas;
- no se usa para tomar decisiones sobre humanos reales.

## Prohibiciones explícitas

E.C.O. no debe:

- diagnosticar enfermedades;
- reemplazar evaluación médica;
- detectar conciencia humana real;
- inferir intención interna de una persona;
- determinar culpabilidad o responsabilidad forense;
- modificar reglas, baseline o umbrales estables sin auditoría.

## Uso correcto

El uso correcto es experimental: observar señales, comparar rutas, medir coherencia, auditar límites y documentar incertidumbre.

## Límite responsable

Gobernanza educativa/experimental. Este documento habilita investigación controlada, no uso clínico, diagnóstico, forense ni modelado de conciencia humana.

# Índice de investigación E.C.O.

## Estado

Estado: experimental.

Clasificación: permitido.

## Propósito

Este índice organiza las piezas de investigación E.C.O. relacionadas con patrones mínimos, Vacuum State y trazabilidad operativa.

## Piezas disponibles

- Documento conceptual: `docs/research/eco-vacio-cuantico-patrones-minimos.md`
- Demo computacional: `scripts/run_eco_vacuum_state_demo.py`
- Trazabilidad: `docs/research/eco-vacuum-state-demo-traceability.md`
- Comando operativo: `make eco-vacuum-state-demo`
- Entrada pública: sección `E.C.O. Vacuum State demo` en `README.md`

## Lectura responsable

Estas piezas son educativas y experimentales. No miden el vacío cuántico real, no usan datos sensibles, no entrenan modelos, no modifican baseline, no recalibran umbrales y no afirman aplicaciones físicas reales.

## Criterio de aceptación

- El índice debe existir.
- Debe enlazar las piezas principales.
- Debe declarar límites responsables.
- Debe ayudar a ubicar el estado de investigación sin mezclarlo con afirmaciones aplicadas.

# Demo mínima de simulación E.C.O.

## Estado

Estado: experimental.

Clasificación: permitido.

## Propósito

Crear una simulación mínima, determinística y verificable del ciclo E.C.O. usando solo datos sintéticos.

## Variables simuladas

- nutrient: recurso sintético disponible.
- signal: activación interna sintética.
- waste: residuo acumulado sintético.
- stability: equilibrio operativo sintético.

## Comando

```bash
make eco-minimal-simulation-demo
```

## Límite responsable

Esta demo no estudia organismos reales, no usa datos sensibles, no entrena modelos, no modifica baseline, no recalibra umbrales y no hace afirmaciones biomédicas aplicadas.

# Alcance de simulación E.C.O.

## Estado

Estado: experimental.

Clasificación: permitido.

## Propósito

Este documento define el alcance seguro para pasar de organización operativa a simulación experimental dentro del proyecto E.C.O.

## Separación responsable

- Estudiar datos: observar, describir y organizar información disponible.
- Simular comportamiento: crear reglas mínimas para representar dinámicas educativas o experimentales.
- Entrenar modelos: ajustar parámetros usando datos; requiere autorización, trazabilidad y control de datos.
- Evaluar resultados: comparar salidas contra criterios definidos sin exagerar conclusiones.
- Generar hipótesis: proponer ideas exploratorias que deben quedar marcadas como hipótesis.
- Hacer afirmaciones aplicadas: declarar efectos físicos, biomédicos o clínicos; queda fuera del alcance actual.

## Alcance permitido ahora

- Crear simulaciones educativas y trazables.
- Usar datos sintéticos o ejemplos mínimos.
- Documentar supuestos, límites y criterios de aceptación.
- Mantener resultados separados de afirmaciones científicas aplicadas.

## Fuera de alcance

- No usar datos sensibles sin propósito, consentimiento y trazabilidad.
- No entrenar modelos con datos sensibles sin autorización clara.
- No modificar baseline estable sin comparación.
- No recalibrar umbrales estables sin auditoría.
- No hacer afirmaciones biomédicas aplicadas.
- No convertir metáforas simbólicas en conclusiones científicas.

## Próximo tipo de simulación sugerida

El próximo avance técnico debería ser una simulación mínima con datos sintéticos, reglas explícitas y salida verificable.

# E.C.O. synthetic demo decision guide

## Propósito

Esta guía explica cómo leer las demos sintéticas de E.C.O. como señales operativas mínimas.
No convierte los resultados en diagnóstico, predicción biomédica ni conclusión aplicada.

## Lectura por demo

| Demo | Qué observa | Decisión operativa permitida | Límite responsable |
|---|---|---|---|
| minimal simulation | Señal mínima procesable | Confirmar que el pipeline puede digerir una entrada básica | Datos sintéticos; sin afirmaciones aplicadas |
| signal balance | Balance entre señales sintéticas | Observar si una entrada domina o se equilibra con otra | Sin extrapolar a organismos reales |
| waste pressure | Presión residual sintética | Detectar acumulación o carga residual simulada | No usar como indicador clínico ni biológico real |
| absorption threshold | Umbral de absorción sintético | Observar cuándo una señal supera un umbral definido | No recalibrar umbrales estables sin auditoría |

## Regla de uso

Usar esta guía para estudiar patrones, comparar comportamiento sintético y decidir próximos pasos de desarrollo.
No usarla para entrenar modelos, procesar datos sensibles, modificar baseline estable ni hacer afirmaciones biomédicas aplicadas.

## Comandos relacionados

- `make eco-validate-synthetic-demos`
- `make eco-synthetic-demos-suite-report`
- `make eco-synthetic-demo-comparison-report`
- `make eco-check-clean`

## Límite responsable reforzado

Esta guía usa solo datos sintéticos; sin entrenamiento; sin datos sensibles; sin modificación de baseline; sin recalibración de umbrales; sin afirmaciones biomédicas aplicadas.
